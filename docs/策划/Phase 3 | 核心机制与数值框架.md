## 二、难度分档系统

### 2.1 难度对属性衰减的影响

所有衰减速率通过**难度系数**统一缩放，只需调整一个参数即可影响全局体验：

```
实际衰减速度 = 基础衰减速度 × 情境修正系数 × 难度系数

难度系数 [PARAM: difficulty_multiplier]：
  简单模式  = 0.6
  中等模式  = 1.0（基准值）
  困难模式  = 1.5
```

### 2.2 自定义难度选项

除三档预设外，提供以下**单项可调参数**，玩家可自由组合：

| 选项名称 | 参数标签 | 可调范围 | 说明 |
|---------|---------|---------|------|
| 饥饿衰减速率 | [PARAM: hunger_decay_rate] | 0.3 - 2.0 | 越高越容易饿 |
| 口渴衰减速率 | [PARAM: thirst_decay_rate] | 0.3 - 2.0 | 越高越容易渴 |
| 体温衰减速率 | [PARAM: warmth_decay_rate] | 0.3 - 2.0 | 越高越容易失温 |
| 体力恢复速率 | [PARAM: stamina_recover_rate] | 0.5 - 2.0 | 越高恢复越快 |
| 庇护所内衰减修正 | [PARAM: shelter_decay_modifier] | 0.1 - 0.8 | 越低庇护所越安全 |
| 死亡惩罚开关 | [PARAM: death_penalty_on] | true/false | 关闭后死亡无读档惩罚 |
| 庇护所外存档 | [PARAM: outdoor_save_on] | true/false | 简单模式默认开启 |

> **设计说明：**
> 三档预设本质上是以上参数的预设组合包。
> 自定义模式让玩家可以精确控制自己想要的体验强度，
> 同时开发阶段策划可以直接修改参数包进行数值测试。

## 四、濒死危机系统

### 4.1 触发条件与效果

| 危机类型 | 触发条件 | 移动速度 | 挖掘效率 | 建造效率 | HP损耗/分钟 |
|---------|---------|---------|---------|---------|-----------|
| 饥饿危机 | 饥饿值=0 | ×0.8 | ×0.7 | ×0.7 | [PARAM: hp_drain_hunger] = -1 |
| 口渴危机 | 口渴值=0 | ×0.7 | ×0.6 | ×0.6 | [PARAM: hp_drain_thirst] = -2 |
| 失温危机 | 体温值=0 | ×0.6 | ×0.5 | ×0.5 | [PARAM: hp_drain_cold] = -3 |
| 三重危机 | 三项同时=0 | ×0.4 | ×0.3 | ×0.3 | [PARAM: hp_drain_triple] = -6 |

### 4.2 危机存活时间估算（中等难度，HP=100）

```
单项危机存活时间：
  饥饿危机  = 100 ÷ 1 = 100分钟
  口渴危机  = 100 ÷ 2 = 50分钟
  失温危机  = 100 ÷ 3 ≈ 33分钟
  三重危机  = 100 ÷ 6 ≈ 16分钟

设计目标：给予玩家足够的反应时间，但不能无限拖延
如需调整压迫感，直接修改对应 hp_drain 参数即可
```

```
单格挖掘时间（秒）= 地层硬度系数 ÷ 工具效率系数

示例（中等难度）：
  初始工具挖S层  = 1.0 ÷ 1.0 = 1.0秒/格  ✅ 流畅
  初始工具挖L1层 = 2.0 ÷ 1.0 = 2.0秒/格  ✅ 略慢，有压迫感
  L1工具挖L1层   = 2.0 ÷ 2.0 = 1.0秒/格  ✅ 恢复流畅
  L1工具挖L2层   = 3.5 ÷ 2.0 = 1.75秒/格 ✅ 略慢，驱动升级
```

## 八、核心循环数值验证（中等难度）

```
早期循环预期（游戏前2小时）：

初始状态：HP=100，饥饿=100，口渴=100，体温=100，体力=100

Step1：地表收集木材（约10分钟）
  体力消耗：-40（挖掘5格×8点）
  饥饿：-20，口渴：-30，体温：-40
  剩余：HP=100，饥饿=80，口渴=70，体温=60，体力=60

Step2：返回庇护所休息（约10分钟）
  体力恢复：+50（庇护所内5点/分钟×10分钟）
  饥饿：-6（2×0.3×10），口渴：-9（3×0.3×10）
  剩余：HP=100，饥饿=74，口渴=61，体温=60，体力=100（上限）

Step3：再次出门收集食物与水（约15分钟）
  饥饿：-30，口渴：-45，体温：-60
  剩余：HP=100，饥饿=44，口渴=16，体温=0 → 触发失温危机警告

Step4：返回庇护所处理资源（约10分钟）
  食用熟食：饥饿+35=79
  饮用过滤水：口渴+40=56
  体温在庇护所内缓慢恢复（供暖炉建成前仍有衰减）

Step5：积累材料，建造供暖炉（约30分钟）
  供暖炉建成 → 体温衰减在庇护所内归零
  ✅ 第一个重要情感节点达成

预期总时长：约90-120分钟
节奏评估：紧张但有节奏，每次出门都有明确目标
          失温危机在早期自然触发，驱动玩家优先建造供暖炉
```
